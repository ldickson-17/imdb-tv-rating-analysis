# IMDb TV Show Rating Analysis

## Overview

This project explores factors associated with IMDb television ratings using regression analysis in R.

The analysis focuses on how variables such as:

- Runtime
- Show age
- Number of IMDb votes
- Number of episodes

relate to average IMDb ratings for television series.

## Tools Used

- R
- tidyverse
- ggplot2
- stargazer
- regression modeling

## Methods

- Cleaned and merged IMDb datasets
- Conducted exploratory data analysis
- Built multiple regression models
- Compared model performance and coefficients
- Visualized relationships between variables

## Key Takeaways

The analysis showed that popularity measures such as IMDb votes and show longevity were associated with differences in television ratings. The project also demonstrated practical applications of regression modeling and data visualization techniques in R.
